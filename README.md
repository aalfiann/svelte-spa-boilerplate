# svelte-spa-boilerplate

Make a svelte spa app compatible with IE11.

Technologies used:

- rollup
- babel
- core-js
- tailwind css using CDN

## Usage

1. Clone this repository
2. Install package
```
npm install
```
3. Run
```
npm run dev
```

### Original project
This is a copy of the official [Svelte Template](https://github.com/sveltejs/template).
